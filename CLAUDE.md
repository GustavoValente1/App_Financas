# Finanças da Casa — instruções para o Claude Code

## Commits e GitHub

Após **qualquer alteração de código** neste projeto, sem exceção e sem precisar ser solicitado:

1. Criar um commit com mensagem descritiva em inglês (conventional commits: `feat:`, `fix:`, `refactor:`, etc.)
2. Fazer `git push origin main` imediatamente após o commit

Nunca deixar alterações sem commit ou sem push ao final de uma sessão de trabalho.
